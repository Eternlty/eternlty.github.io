什么是 markdown-it？#

markdown-it 是一个功能强大、可扩展的 Markdown 解析器，它速度快、符合标准，并且支持丰富的插件系统。通过配置各种插件，我们可以扩展 Markdown 的语法，实现更多高级功能。

本文将详细介绍在我们的 Hexo 博客中已配置的 markdown-it 增强语法插件，帮助你更好地编写博客文章。

已配置的增强语法插件#

在我们的博客中，已经配置并启用了以下 markdown-it 扩展插件：

  1. markdown-it-abbr - 支持缩写
  2. markdown-it-footnote - 支持脚注
  3. markdown-it-ins - 支持下划线
  4. markdown-it-mark - 支持标记文本
  5. markdown-it-sub - 支持下标
  6. markdown-it-sup - 支持上标

下面将详细介绍每种增强语法的使用方法和效果。

常用的 markdown-it 插件推荐#

除了我们已经安装的插件外，以下是一些常用的 markdown-it 插件推荐：
